# Contact Book

A simple command-line Contact Book application built with Python and Object-Oriented Programming (OOP).

This project allows users to:

+ Add contacts

+ View all contacts

+ Search contacts

+ Update contact information

+ Delete contacts

+ Save contacts permanently using JSON


## Features

+ Object-Oriented Design

+ JSON File Persistence

+ Partial Name Search

+ Update Existing Contacts

+ Delete Contacts

+ Clean CLI Interface

+ Modular Project Structure


## Project Structure

```text
Contact_Book/
│
├── main.py
├── contact_book.py
└── contacts.json
```

## Technologies Used

+ Python 3

+ JSON

+ Object-Oriented Programming (OOP)

## How It Works

The application stores contacts inside a JSON file (contacts.json).

When the application starts:

+ Contacts are loaded from the JSON file

+ Users can perform CRUD operations

+ Changes are automatically saved
